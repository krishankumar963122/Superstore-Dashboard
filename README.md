# Superstore-Dashboard
Project Title : Superstore Sales Analytics Dashboard 


 Short Description / Purpose : The Superstore Sales Dashboard is an interactive Power BI project designed to analyze and visualize retail business performance using the Superstore dataset. The dashboard helps businesses identify trends, evaluate performance, and make data-driven decisions.

 
Tech Stack :
       Power BI
       DAX (Data Analysis Expressions)
       Power Query
       Data Visualization
       Microsoft Excel / CSV Dataset
       

Features / Highlights : 
       Dynamic metric selection using buttons/slicers
       Category-wise quantity analysis
       State-wise performance tracking
       Customer Segment Analysis
       Region-wise distribution analysis
       Shipping Mode Insights
       Year-based filtering (2018–2021)

       

An interactive Power BI dashboard that analyzes Superstore sales data across categories, regions, states, customer segments, and shipping modes to deliver actionable business insights through dynamic visualizations and KPI tracking.
